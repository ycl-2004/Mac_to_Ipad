# YC Cast v8 Release Notes

YC Cast v8 focuses on the Mac-to-iPad extended display workflow.

## Highlights

- User-facing app name and packaging are now YC Cast.
- New connected-device cleanup: duplicate records are merged, hidden devices can be shown, and saved devices can be removed.
- New default display placement: the virtual display is placed to the right of the Mac display by default.
- New display placement setting: choose right, left, above, or below before connecting.
- New iPad disconnected state: the iPad shows "Device disconnected" when sharing stops or the network drops.
- New larger-text HiDPI resolution: `1024 x 768 HiDPI (Larger Text)`.

## Build Notes

- Mac local builds use `env SIGN_IDENTITY=- ./make_app.sh`.
- Public distribution should use your own Developer ID certificate and notarization credentials.
- Generated app, DMG, IPA, zip, and `dist/` artifacts are release assets, not committed source files.
